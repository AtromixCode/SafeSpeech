Install node.js https://nodejs.org/en/download/
The project requires running two instances, thus have one side running the serve and the other running the start (front end + back end).

Additionally access our project via http://safespeech.onthewifi.com/#/

# front-end

## Project setup
 add all the required libraries and lint for your file system
```
cd front-end
npm install
npm run lint
```

### Compiles and hot-reloads for development
run the project front end
```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```

### Customize configuration

See [Configuration Reference](https://cli.vuejs.org/config/).

# Back-end

## Project setup
install requirements for back end and run the back end.
```
cd back-end
npm install
npm run start
```
